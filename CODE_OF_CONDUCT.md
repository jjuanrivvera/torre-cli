# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks
are not tolerated. Report unacceptable behavior to the maintainer via a private channel
(see [SECURITY.md](SECURITY.md) for how to reach out privately). Maintainers may remove,
edit, or reject contributions that violate this code.
