## torre alias set

Create or update an alias

```
torre alias set <name> <expansion> [flags]
```

### Examples

```
  torre alias set inbox "mail list --folder inbox"
  torre alias set week "calendar events"
```

### Options

```
  -h, --help   help for set
```
